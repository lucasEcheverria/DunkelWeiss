¿Cuál es el mejor IDE para desarrollo web en 2026?

Quiero abrir debate sobre los IDEs más usados actualmente. Llevo años con IntelliJ pero últimamente estoy viendo mucha gente pasarse a VS Code o incluso Neovim con configuraciones avanzadas. ¿Qué usáis vosotros y por qué? Me interesan especialmente opiniones sobre rendimiento, plugins y experiencia general.

VS Code sin duda
Llevo 4 años con VS Code y no lo cambio por nada. El ecosistema de extensiones es brutal, arranca en segundos y con las extensiones de IntelliSense tienes autocompletado casi al nivel de IntelliJ. Además es gratis y open source. Para web no necesitas más.

IntelliJ sigue siendo el rey para JVM
Como autor del hilo, tengo que defender IntelliJ. Para Spring Boot + Java/Kotlin no hay nada que se le acerque. El refactoring, la integración con Gradle/Maven, el debugger... Es cierto que consume más recursos, pero si tienes un equipo decente, merece la pena. Para web pura quizá VS Code, pero para fullstack con backend JVM, IntelliJ sin duda.
